# Hypixel-API-Bedwars-Discord-Bot
## Hypixel API Discord bot that has a few bedwars, skywars, misc, stat commands.

## Commands: 

>!bwgames (bw means Bedwars)
>
>!bwwins
>
>!bwwlr (win/loss ratio)
>
>!level
>
>!rank
>
>!swgames (sw means Skywars)
>
>!swwins
